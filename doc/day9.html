Chapter 1. Day 9

Installing the Bootstrap Gem

Bootstrap is not rails specific; however, there is a gem for this. We will be using the "bootstrap-sass" gem.

Procedure 1.1.  Install the "Bootstrap" front end framework

  1. Navigate to the Gemfile in the project root

    Add the following line to the file:

    gem 'bootstrap-sass'

    Note that a comma and version number is not specified. The above line of code should be enough

  2. Navigate to the project root in the terminal and run the command: $ bundle install

    Go to the app and refresh the page and get an error

    This is because the server needs to be reinitialized because everything got frozen when we ran bundle.

  3. Include this gem in the "Asset Pipeline" of the application

    Navigate to ./app/assets/stylesheets/ and create the file bootstrap_and_customization

    In that file, add the line of code:

    @import 'bootstrap';

  4. Run $ rails server to re-initialize the server

    If there is an error then run $ bundle install again and see if any new gems get pulled in.

    At this point running $ rails server should work.

  5. Load up the app and see that the Bootstrap customization has taken effect by noting the different styling of the page content.
