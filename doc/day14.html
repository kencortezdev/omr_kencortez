Chapter 1. Day 14

Using the Very Popular Devise Gem for Adding Users

Procedure 1.1.  Installing devise

  1. Read about the devise gem on the GitHub page

    Add to the gem file: gem 'devise', '~> 3.1.0.rc2'

    Run $ bundle install because we've updated the Gemfile

  2. Run the command % rails generate devise:install

Procedure 1.2.  Devise Setup Steps

Devise setup notes are numbered and are displayed at the end of the previous procedure

  1. Set up the URL defaults for development and prodocution

    The code that is displayed in the terminal can simply be copied and pasted into the paths specified after running the installation

  2. Add flash message embedded ruby code in the application.html.erb

    Don't forget to add the end tag at the end

    Also note that the '=' signs are gone for the flash tag. This is a subtlety of embedded Ruby: it is being run, but not being shown on the page

  3. Add a content_tag tag inside of the flash element

    The tutorial is a bit cryptic about it and doesnt go into detail about what the embedded Ruby really does here

  4. Add code inside of the class application in the application.rb file for Heroku support (copy and paste again)

  5. Run the $ rails g devise:views generate command to generate the views

    If successful, there should be a new folder nested in the views folder of the project
