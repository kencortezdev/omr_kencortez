Procedure 1.1. Set Up a New Rails Project

  1. Go to the folder where you want your rails project

  2. I have chosen C:\one_month_rails\

  3. Run the command: $ rails new omr_kencortez

  4. The setup will also run bundle insall to fetch needed gems
  5. In the end there should be a named project folder with a bunch of directories in that file

  6. Move into the project folder and run the command $ rails server

    This starts a little server called WEBrick and runs in the browser

  7. Access by navigating to http://localhost:3000/
