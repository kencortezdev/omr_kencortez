Procedure 1.1.  Creating Navigation Links Using the Application Layout

Commonly, in the header of a page is a navigation bar that is consistent across multiple pages. This section tells you how to maintain that using "Layouts".

  1. Go to ./app/views/layouts/ and open application.html.erb

    Add the embedded Ruby below in the "body" tag (just above where it says "yield") to the file:

    <%= link_to "Home", root_path %>

    <%= link_to "About", about_path %>

  2. Navigate to the app and see that the navigation bar (with functional links) appear on both the "Home" and "About" pages.
