Chapter 1. Day 4

Version Control With Git

Procedure 1.1.  Commit with Git

  1. Set up git in the command line: $ git config --global user.name "Ken Cortez"

  2. Verify that this is set by qurerying with the same command (without quotes) $ git config --global user.name

  3. Set up email and query email the same way with $ git config --global user.email "kenmcortez@yahoo.com"

    Note that I am using my Yahoo email!

  4. Now cd into the project folder and run the command $ git init

    This obviously initializes git inside of our project

  5. stage all files in the root dir for addition by entering

    $ git add .

  6. Then commit all files with messages "m" applied to all "a" committed

    $ git commit -am "initial commit"

    This is just a standard for the initial commit

Procedure 1.2. Recover Deleted Entries

  1. Type $ git add . but do NOT commit

  2. Type $ git checkout -f which brings back everything
