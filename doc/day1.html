Chapter 1. Day 1

Installing Rails

Procedure 1.1. Install Rails

  1. Go to http://www.installrails.com/

  2. Downloaded Rails Installer 3.2.2

    Bundles Ruby 2.2.4p173

    Bundles Rails 4.2.4

  3. Once done installing, verify the following:

    $ rails -v

    $ ruby -v

  4. The installer should have added this to PATH
